# Logical Fallacies

A tutor that teaches you to spot bad arguments — not by handing you a list of definitions, but by showing you flawed reasoning and asking you to figure out what's wrong before giving it a name.

**What it teaches:** 12 common logical fallacies, how to identify them in news headlines and social media, how to distinguish between similar fallacies, and how to evaluate whether an argument is genuinely sound or just persuasive.

**What it demonstrates:** How ScaffoldAI handles **analytical reasoning** — a skill that requires pattern recognition, judgment, and the ability to explain *why* something fails, not just label it.

---

## Setup

1. Open [Claude](https://claude.ai) and create a new Project.
2. Open the `tutor-instructions.md` file in this folder.
3. Copy the entire contents and paste it into your Project's **Custom Instructions**.
4. Start a new conversation within the project.

That's it. The tutor will guide you from there.

---

## Suggested First Prompts

Not sure where to start? Try one of these:

- **"I want to get better at spotting bad arguments online."** — A natural starting point. The tutor will begin with what makes an argument an argument, then start showing you how they break.

- **"I know what a straw man is, but I couldn't name many other fallacies."** — Gives the tutor a baseline. It'll check your understanding of straw man, then build outward to related fallacies.

- **"I feel like I can tell when something's wrong with an argument, but I can't always explain why."** — This is exactly the gap the tutor is built for. It'll help you move from gut feeling to specific diagnosis.

- **"Show me some arguments and I'll tell you what's wrong with them."** — Jumps straight to practice. The tutor will start with clear examples and ramp up to trickier cases where multiple fallacies overlap.
