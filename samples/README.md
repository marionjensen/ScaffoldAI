# Sample Tutors

These three tutors demonstrate ScaffoldAI across different types of learning. Each one is ready to use — just copy the instructions into a Claude Project and start a conversation.

| Sample | Topic | What It Demonstrates |
|---|---|---|
| [Impressionism & Post-Impressionism](impressionism/) | Key movements, artists, techniques, and the philosophical shifts between them | Teaching **conceptual knowledge** — building understanding of ideas and connections |
| [Photography Exposure Basics](photography/) | Aperture, shutter speed, ISO, and depth of field | Teaching an **applied skill** — learning settings that interact with each other |
| [Logical Fallacies](logical-fallacies/) | 12 common reasoning errors and how to spot them in real arguments | Teaching **analytical reasoning** — identifying flawed logic in the wild |

Each sample includes setup instructions, the complete tutor instructions (Layer 1 + Layer 2 combined), and suggested first prompts to get you started.
